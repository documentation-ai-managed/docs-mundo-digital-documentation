## Upgrade to Latest Version

Follow these steps to update your Mundo Digital Documentation site.

<Steps>
  <Step title="Backup" icon="database">
    Create a backup of your current `docs/` folder.

    ```bash
    cp -r docs docs-backup
    ```
  </Step>
  <Step title="Update Package" icon="download">
    Install the latest version using your package manager.
  </Step>
  <Step title="Rebuild" icon="refresh-cw">
    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm run build
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```bash
        yarn build
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Deploy" icon="upload-cloud">
    Push changes to your hosting provider.
  </Step>
</Steps>
